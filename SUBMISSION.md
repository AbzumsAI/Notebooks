# Assignment Submission Guide

I use this flow to keep homework links easy to check.

1. Make a repo in your own GitHub account for the solved notebook.
2. Keep the original assignment name in the file name.
3. Fill the student info cell if the notebook has one.
4. Run the notebook from top to bottom before sharing it.
5. Push the notebook to your repo.
6. Send the GitHub link to the course email before the deadline.
7. Check that the link is public or shared with the instructors.

Before sending a link, remove API tokens, private links, and personal data that is not needed for grading.
